# 🕒 Jam App - Mobile Programming UAS

Aplikasi jam multifungsi yang dibangun menggunakan **Flutter** sebagai tugas akhir semester 4 di **Universitas Islam Madura**. Aplikasi ini dirancang dengan antarmuka yang bersih dan elegan, mengutamakan pengalaman pengguna yang intuitif.

## 🚀 Fitur Utama
Sesuai dengan rencana pengembangan, aplikasi ini mencakup empat fitur utama:
*   **Alarm**: Pengaturan waktu bangun dengan antarmuka toggle yang modern.
*   **Jam Dunia**: Memantau waktu di berbagai zona waktu (Jakarta, London, dll).
*   **Stopwatch**: Penghitung waktu presisi dengan fitur putaran (lap).
*   **Pewaktu (Timer)**: Penghitung mundur dengan pilihan durasi cepat.

## 🛠️ Teknologi yang Digunakan
*   **Framework**: Flutter
*   **Bahasa Pemrograman**: Dart
*   **Package**: `intl` (untuk pemformatan waktu dan tanggal)

## 📖 Cara Penggunaan
1.  **Instalasi**: Pastikan Flutter sudah terinstal di perangkat Anda.
2.  **Clone/Download**: Ambil kode dari repository ini.
3.  **Ambil Dependencies**: Jalankan perintah berikut di terminal:
    ```bash
    flutter pub get
    ```
4.  **Jalankan Aplikasi**:
    ```bash
    flutter run
    ```

## 📈 Proses Pengembangan (Git)
Proyek ini dikembangkan secara bertahap sesuai ketentuan dosen:
*   Inisialisasi Proyek & Git.
*   Pembuatan Blueprint desain.
*   Implementasi navigasi bawah (Bottom Navigation).
*   Penambahan fitur Jam Dunia, Stopwatch, Alarm, dan Timer secara berkala.
*   Perbaikan layout dan penanganan overflow.
